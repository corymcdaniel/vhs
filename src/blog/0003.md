*I've been doing a few experiments with Claude Code agents, this is one of them.*

## Drink / Drive / University

<a href="https://map.corymcdaniel.com/" target="_blank" rel="noopener noreferrer">map.corymcdaniel.com</a>

I wanted to see how easy it would be to instruct Claude to build a map based site that has a few different layers of data. Something on the news caught my ear and so I tried to get a map showing alcohol consumption rates and an average scoring of universities.

I have no agenda around this choice, it just seemed interesting and likely data easy to do.

### The Prompt

*I asked Claude for the initial prompt and a summary of the process. It gave me the following:*

**Initial Prompt**
"I'd like to find the average GPA for every university in the United States as well as the average drinking consumption by state, then plot the GPA showing University Name, GPA, and average alcohol consumption on the pin. If possible, color the States Green (good) ranging to Red (worst) with good being the lowest alcohol consumption."

### How It Went

The idea evolved quickly. Within the first few follow-ups, GPA was dropped (no clean bulk dataset exists outside of paywalled sources), replaced by a composite academic score using SAT average, ACT midpoint, and graduation rate from the U.S. Department of Education's College Scorecard. The map shifted from a simple pin plot to a full choropleth + marker hybrid with a sidebar panel.

### Data Sourcing

Data sourcing was the biggest challenge. Three separate pipelines were needed:

- **Alcohol** — NIAAA Surveillance Report #122, a fixed-width .txt file spanning 1970–2023. Parsing it required understanding a very specific column layout to extract 2023 per-capita ethanol consumption by state.
- **Universities** — The College Scorecard API was the first approach, but DNS resolution failed in the build environment. The fix was switching to a 24MB bulk ZIP download (~1,900 four-year institutions after filtering).
- **Fuel consumption** — EIA SEDS Table F10 motor gasoline data, combined with 2023 Census Bureau population figures to compute per-capita gallons by state.

### Visual Design

Visual design required iteration. The initial alcohol gradient was technically correct but perceptually flat — state scores clustered between 0.1 and 0.4, making the green-to-red range hard to see. A linear stretch combined with gamma correction (γ=0.58) solved it. Academic tiers got their own icon system: a gold star for elite schools, diamond for high, filled circle for mid, outline circle for lower — all sized and z-indexed so elite pins appear on top.

A second data layer was added mid-project. Vehicle fuel consumption wasn't in the original scope, but it was a natural addition once the state choropleth infrastructure was in place. A pill toggle (🍺 Alcohol / ⛽ Fuel) lets you switch the entire map layer without a page reload.

### The Stack

React + Vite, Leaflet via react-leaflet, CSS Modules + SCSS, no TypeScript, no component libraries. The finished build deploys as a fully static site on Netlify — all data lives in public/data/ as pre-generated JSON, so there are no API keys, no backend, and no runtime costs.

Total scope: ~1,900 university pins, 51 state polygons, two switchable choropleth layers, a sidebar with academic and public health data, and a one-time Node.js data pipeline script that can be re-run to refresh the underlying datasets.
